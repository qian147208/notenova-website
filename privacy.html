隐私政策

版本日期：2026 年 6 月 9 日 | 应用版本：1.0

🔒 核心承诺：灵芯 采用"本地优先"架构。您的所有数据存储在您的设备本地，不会上传至我们的任何服务器。我们不收集、不存储、不分享您的任何个人数据。

一、我们收集的信息

我们几乎不收集任何信息。具体说明如下：

  • 创作数据：您在灵芯 中录入的产品信息、生成的文案内容、历史记录、灵感收藏等，全部存储在您设备的本地存储中，不会上传至任何服务器。
  • 登录凭证：如使用账号登录功能，您的密码通过 Keychain 加密存储在您的设备上，我们不会在服务器上存储您的密码。
  • API 配置：如您配置了第三方 AI 服务（如 OpenAI、DeepSeek 等）的 API Key，该密钥仅存储在您的设备本地，不会上传至我们的服务器。
  • 设备信息：我们不会收集您的设备 IMEI、UDID、MAC 地址、通讯录、相册、位置信息或其他设备标识符。

二、我们如何使用您的信息

由于我们几乎不收集任何信息，因此不存在数据使用问题。您的创作数据仅用于在您设备上生成小红书笔记内容。

如您配置了第三方 AI 服务，您的 API Key 和输入内容将直接发送至第三方服务（如 OpenAI、DeepSeek 等），其数据处理方式受第三方各自的隐私政策约束。

三、我们分享的信息

我们不会向任何第三方分享、出售、出租或转让您的个人信息。

我们不会使用任何第三方分析 SDK（如 Firebase、Adjust 等），不会使用任何广告 SDK，不会进行任何形式的数据追踪或用户画像。

四、数据存储与安全

  • 所有创作数据存储在您的设备本地，采用 SwiftData 框架管理。
  • 登录凭证通过 iOS Keychain 加密存储。
  • 我们没有任何服务器存储您的数据。
  • 如您卸载应用，存储在本地设备上的所有数据将被清除。

五、第三方服务

灵芯 可能通过您自行配置的 API Key 访问第三方 AI 服务。这些服务由第三方运营，受其隐私政策约束。我们建议您在使用前阅读相关第三方的隐私政策。

六、儿童隐私

灵芯 不面向 13 岁以下儿童提供。我们不会故意收集儿童的个人信息。如发现我们无意中收集了儿童信息，请立即联系我们删除。

七、隐私政策的更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内或应用商店页面公布。重大变更时，我们会通过应用内通知告知您。

八、您的权利

  • 您有权随时在应用内查看、编辑或删除您的创作数据。
  • 您有权卸载应用以清除所有本地存储的数据。
  • 如您对我们处理个人数据的方式有任何疑问或要求，请通过以下方式联系我们。

九、联系我们

如您对本隐私政策有任何疑问、意见或建议，或希望行使您的隐私权，请通过以下方式联系我们：
