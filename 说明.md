## 🚀 部署步骤

### 1. 创建GitHub仓库
1. 打开 [github.com/new](https://github.com/new)
2. 仓库名：`redbookrefill-website`
3. 设为 Public
4. 点击 "Create repository"

### 2. 运行部署脚本
```bash
cd 官网
bash 部署脚本.sh
```

### 3. 开启GitHub Pages
1. 打开你的 GitHub 仓库页面
2. 点击 **Settings** → **Pages**
3. **Source** 选择 `main branch`，根目录
4. 等待 1-2 分钟，网站即可访问

## 📋 获取URL

部署成功后，你的 URL 格式为：

| 页面 | URL |
|------|-----|
| 首页 | `https://<你的用户名>.github.io/redbookrefill-website/` |
| 技术支持 | `https://<你的用户名>.github.io/redbookrefill-website/技术支持.html` |
| 隐私政策 | `https://<你的用户名>.github.io/redbookrefill-website/隐私政策.html` |

## 📱 在App Store Connect中使用

填写App Store Connect时：
- **技术支持URL**：填写首页URL
- **隐私政策URL**：填写隐私政策URL

## 📝 需要修改的地方

在 `首页.html` 中搜索以下占位符并替换：
- `https://apps.apple.com/app/你的appid` → 你的App Store链接
