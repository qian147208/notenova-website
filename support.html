技术支持

遇到问题了？这里有一些常见问题和解决方法。如果还没找到答案，欢迎联系我们。

使用指南

如何生成小红书笔记？

1. 在「产品库」中添加你的产品信息（名称、卖点、目标人群等）
2. 进入「生成」页面，选择产品或输入关键词
3. 选择笔记类型（信息流广告 / 搜索优化 / 品牌故事 / 带货转化）
4. 选择风格和模式（快速 / 高质量）
5. 点击生成，即可获得标题、正文、标签和配图建议

如何编辑生成的文案？

生成结果后，你可以：
• 对标题、正文、标签分别点击「重写」或「换一批」
• 长按文本划选需要修改的部分，弹出局部重写选项
• 直接编辑文本内容
• 一键复制全部或直接分享到小红书

Guest 模式限制是什么？

作为未登录用户（Guest），你可以免费体验 3 次生成功能。登录后即可无限使用。

AI 配置

如何配置 AI 服务？

1. 进入「设置」页面
2. 在「AI 配置」中填写你的 API Key
3. 支持的服务：
  • OpenAI 兼容 API（如 ChatGPT）
  • DeepSeek
4. API Key 存储在设备本地（Keychain 加密），不会上传

快速模式和高质量模式有什么区别？

• 快速模式：生成速度快，适合日常批量创作
• 高质量模式：生成内容更精细，适合重要笔记创作
可在生成页面右上角切换

产品管理

如何管理产品库？

在「产品库」页面：
• 添加新产品：填写产品名称、核心卖点、目标人群、使用场景
• 上传产品照片：最多 5 张
• 上传风格参考图：最多 5 张
• 编辑或删除已有产品
所有数据存储在设备本地

灵感板怎么用？

灵感板用于收藏和分类管理你喜欢的文案片段：
• 在生成结果页收藏喜欢的标题、正文或标签
• 灵感分为三类：关键词、文案片段、灵感语录
• 生成笔记时可以快速调用灵感素材作为上下文

隐私与安全

我的数据安全吗？

非常安全。灵芯 采用本地优先架构：
• 所有创作数据存储在设备本地
• 登录凭证通过 Keychain 加密
• 没有服务器存储你的任何数据
• 不含第三方 SDK、追踪器或广告 SDK

卸载 App 会丢失数据吗？

是的。由于所有数据存储在设备本地，卸载应用后数据会被清除。建议重要笔记做好备份（可复制粘贴到备忘录）。
